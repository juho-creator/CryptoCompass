# 加密指南

CryptoCompass 是一个旨在指导加密货币投资者的一体化应用程序。它提供以下功能：

- 市场情绪分析
- 实时比特币交易查询
- 市值变化检查
- 最新新闻获取
- 交易日志撰写

**要下载 CryptoCompass，请[点击这里](https://play.google.com/store/apps/details?id=com.juhooray.crypto_analysis&hl=en_US)。**
