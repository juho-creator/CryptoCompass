# Crypto Compass

Crypto Compass는 암호화폐 투자자를 안내하기 위한 올인원 앱으로, 다음과 같은 기능을 제공합니다:
- 시장 심리 분석
- 실시간 비트코인 거래 조회
- 코인별 시가총액 변화 검토
- 최신 암호화폐 뉴스
- 매매일지 작성

**Crypto Compass를 다운로드하려면 [여기를 클릭하세요](https://play.google.com/store/apps/details?id=com.juhooray.crypto_analysis&hl=en_US).**
